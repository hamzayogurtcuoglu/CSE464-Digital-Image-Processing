Compilation
------------------------------------------------------------
1- Way) 
you can compile the source codes with CMakeList.txt.
first command in project directory -> # cmake . 
second command -> # make

2-Way)
If you dont want to use cmake. 
you can compile with following command this code.

# g++ main.cpp -lpng -o exe
------------------------------------------------------------

That accepts a digital image as input argument,
 performs path opening on it and returns the 2 results path opening image. 
(Note : Please put just png images)
-------------------------------------------------
Execute

./exe image1.png

./exe  image2.png

-------------------------------------------------


1-) The Execution time varies depending on the image 
2-) You will take PathResult1.png and PathResult2.png
3-) You can use any picture found.
